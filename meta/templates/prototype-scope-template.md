# Prototype Scope Template

Used when defining the scope of a game prototype.
Clarify the hypothesis to validate and keep it minimal.

## Frontmatter Structure

```yaml
---
id: proto-{slug}
type: prototype_scope
status: raw | suggested | candidate | edited | canon
title: ""
game_concept_ref: ""
core_loop_ref: ""
tags: []
created: YYYY-MM-DD
updated: YYYY-MM-DD
---
```

## Body Structure

### Hypothesis to Validate
What this prototype aims to verify. Use the format: "If [condition], then [expected outcome]."

### Included Elements
What must be included in the prototype.
- [ ] Element 1
- [ ] Element 2

### Excluded Elements
What is intentionally left out of this prototype, with reasons for exclusion.

### Success Criteria
Criteria for determining that this prototype is a "success."

### Production Scope
- **Estimated Time**:
- **Required Resources**:
- **Format**: Paper prototype | Digital | Hybrid

### Test Plan
Who will test it and how.

### Notes
Free-form.
