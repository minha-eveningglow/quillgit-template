# Game Concept Template

소설 세계관을 기반으로 한 게임 컨셉을 정의할 때 사용.
줄거리 재현이 아닌 플레이 경험 재설계에 초점.

## Frontmatter 구조

```yaml
---
id: game-{slug}
type: game_concept
status: raw | suggested | candidate | edited | canon
title: ""
genre: ""
platform: ""
player_count: ""
session_length: ""
tags: []
created: YYYY-MM-DD
updated: YYYY-MM-DD
source_canon_refs: []
---
```

## 본문 구조

### 한줄 컨셉
이 게임을 한 문장으로.

### 원작과의 관계
소설의 어떤 요소를 게임화하는가. 무엇을 변환하고 무엇을 버리는가.

### 플레이어 경험
플레이어가 이 게임에서 하는 핵심 행위는 무엇인가.

### 핵심 감정
플레이어가 느끼길 원하는 감정.

### 장르/형식
게임 장르, 플랫폼, 인원, 시간 등.

### 차별점
기존 유사 게임과 무엇이 다른가.

### 검증 질문
이 컨셉이 작동하는지 확인하기 위한 질문들.

### 메모
자유 형식.
